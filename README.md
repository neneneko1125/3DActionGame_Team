ジャンル：3Dアクションゲーム  
制作目的：今まで2Dゲームを制作していたが、3Dにも挑戦するため  
苦労した点：プレイヤーのデータの管理　プレイヤーの元データから、ゲーム中に使うデータを作成し、それを適切に他のクラスに配布すること、非エンジニアの人でもプレイヤーのステータスを調整できるように、ステータスの項目をまとめたScriptableObjectの作成  
工夫した点：攻撃が単調にならないようにする、かつ誰にでも操作がわかるようにするために、ボタン一つで数種類の攻撃をできるような仕組みにした。  
使用エンジン：Unity6  
バージョン：6000.4.4f1  
使用言語：C#  
プレイ動画：https://youtu.be/IKIE0rIXYqg  

このゲームは二人で作成しました。自分のソースコードはAssets→_LVFolder→Scripts～に収納されています。
